🔑 كيفية الحصول على مفتاح Gemini API

⚠️ مهم: المفتاح الموجود حالياً في المشروع هو مفتاح تجريبي ولن يعمل. يجب الحصول على مفتاح حقيقي من Google.
1
افتح موقع Google AI Studio

انقر على الرابط التالي:

🌐 فتح Google AI Studio
2
سجل الدخول

استخدم حساب Google الخاص بك (Gmail)

إذا لم يكن لديك حساب، أنشئ واحداً مجاناً

3
أنشئ مفتاح API

انقر على زر "Create API Key"

اختر مشروع موجود أو أنشئ مشروع جديد

4
انسخ المفتاح

سيظهر لك مفتاح يبدأ بـ AIzaSy...

انسخه بالكامل (حوالي 39 حرف)

5
ضع المفتاح في المشروع

افتح ملف .env.local في مجلد المشروع

استبدل المفتاح القديم بالمفتاح الجديد:

VITE_API_KEY=AIzaSyC4R8x9Tn2Lp3Km5Qw7Yz1Vb6Nj8Hg0Fd2

ملاحظة: ضع مفتاحك الحقيقي بدلاً من المثال أعلاه

6
أعد تشغيل المشروع

في Terminal، أوقف السيرفر (Ctrl+C)

ثم شغله مرة أخرى:

npm run dev
7
حدّث المتصفح

اضغط F5 أو Ctrl+R لتحديث الصفحة

افتح: http://localhost:3001

🎉

تم بنجاح!

الآن يمكنك استخدام المنصة التعليمية بكامل ميزاتها

🔒 تحذير أمني:
  • لا تشارك مفتاح API مع أحد
  • لا ترفع ملف .env.local إلى GitHub
  • المفتاح مجاني ولكن له حدود استخدام
